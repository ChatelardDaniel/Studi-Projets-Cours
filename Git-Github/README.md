# Projets-Cours-Studi

## Projet-Studi-Git-et-Github

Mon premier projet Git avec Studi.

Ce repository a été créé lors d'un projet étudiant pour tester Git et ses fonctionnalités.

## 1 - Créer un repository sur Github

Allez sur le site Github, connectez-vous sur votre compte.
Cliquez sur 'new' et donnez un nom à votre repository,
choisissez les options 'public ou private' et initialisez-le avec un fichier 'README.md'.

Modifier le README.md en cliquant sur le crayon à droite, puis faites un commit en cliquant sur le bouton vert 'Commit changes' en haut.

Voir les commits en cliquant sur 'code' en haut à gauche, puis sur 'commit'.

## 2 - Cloner le repository local

Ouvrez votre terminal, naviguez vers le répertoire où vous souhaitez cloner le repository.
Sur la page Github, cliquez sur le bouton 'code' et copiez l'URL du repository.
Dans votre terminal, executez la commande suivante 'git clone' avec l'URL que vous avez copiée, cela créera une copie locale du repository de Github sur votre machine.

```javascript
git clone URL de Github

Exemple:
git clone https://github.com/ChatelardDaniel/Projet-Studi-Git-et-Github.git
```
